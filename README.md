# QGIS
Geospatial analysis and mapping projects created using QGIS. This work includes spatial data processing, terrain analysis using DEM, and preparation of thematic maps such as slope, aspect, NDVI, and land use/land cover (LULC). The projects also involve vector data digitization, feature extraction from OpenStreetMap, buffer and distance analysis, coordinate system management, reprojection, and raster–vector processing. The repository reflects experience in completing research and freelance-style GIS tasks while continuously learning and adapting to new tools and workflows in QGIS.
